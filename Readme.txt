файл functions - в нем первые две функции нужны только для того, 
чтобы сформировать json файл с адресами квартир, которые будут в последствии парситься.
последняя функция - get_flats_info - парсит конкретную квартиру и записывает данные в 
базу.

data.db - сама база, посмотреть можно с помощью DB Browser for SQLite.

файл main.py - раскомментировать закомментированное для того, чтобы создать json, 
в таком виде вызывает только одну функцию - get_flats_info.

файл everydayrun - для запуска функции get_flats_info каждый день

файл dbtests: 
- def average_price - показывает среднюю цену 1 кв метра всех домов.
- average_price_all - среднюю цену 1 кв метра по каждому адресу на дату.
- searching_flat_for_client - пример поиска квартиры для клиента по заданные критериям,
в данном случае - площадь квартиры и цена.

По поводу api, которая ищет координаты домов - так как в объявлениях у новостроек указан
не точный адрес, работает на слишком точно. Но по вторичке - находит хорошо.
